웹 페이지 작성

 쇼핑몰
  -메인페이지
  -회원가입 페이지
  -상품목록
  -상품 상세 페이지

 요구사항 정의
  기능리스트
   - 메인 페이지
       네비게이션 바에 홈버튼, 회원가입 버튼, 제품 목록 버튼을 제작
       홈 버튼 - 누르면 홈화면으로 귀환
       회원가입 버튼 - 회원가입 페이지로 전환
       제품 목록 버튼 - 제품 목록이 나오게 제작
   
   - 회원가입 페이지
       이메일, 이름, 아이디, 주소만 입력하는 창 제작
       4가지 요소 모두 필수 입력 요소로 제작
 
   - 상품목록
       전체 판매하는 상품 목록 보여주기
   
   - 상품 상세 페이지
       상품 상세 설명 및 사진 첨부
